# Compliance Notes

SlideBridge Core is designed as a clean public core for research and algorithm development.

- It does not include proprietary SDKs.
- It does not include proprietary format implementations.
- It is not for clinical diagnosis.
- It does not upload slide data or clinical data.
- It does not represent, endorse, or certify any scanner vendor.
- Heatmap and score overlays are algorithm debugging visualizations only.
- Example annotations are synthetic.
- Do not upload patient-identifiable annotations to issues or examples.
- It does not include proprietary annotation format implementations.
- Annotation overlays and patch labels are research/debugging visualizations only.

Private readers should be distributed separately under appropriate licenses.
