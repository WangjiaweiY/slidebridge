# SlideBridge Core v0.2.0

## Highlights

- Synthetic demo image and overlay for quick start
- WSI metadata inspection
- Local browser viewer
- Patch coordinate overlay
- Score / attention heatmap overlay
- Patch export
- QC report
- CSV / NPY / H5 / JSON / optional PT/PTH patch table support
- GitHub Actions CI

## Important Notice

- Research and algorithm development only
- Not for clinical diagnosis
- No proprietary vendor SDKs
- No proprietary vendor format implementations
- No patient data included

## Quick Start

```powershell
pip install git+https://github.com/WangjiaweiY/slidebridge.git
slidebridge create-demo --out outputs\demo_slide.png
slidebridge sample-patches outputs\demo_slide.png --out outputs\demo_coords.csv `
  --count 200 --with-scores
slidebridge render-overlay outputs\demo_slide.png --patches outputs\demo_coords.csv `
  --out outputs\demo_overlay.png
slidebridge inspect outputs\demo_slide.png
```

## Known Limitations

- Viewer uses OpenSeadragon CDN
- Heatmap overlay still uses DOM overlay and may be slow for very large patch counts
- Browser visual regression is not automated
- Large WSI performance depends on reader/backend
